src/maskers/phone.ts

maskPhone
maskPhone(phone: string, options: Pick)

Mask a phone number.

  • Preserves "+" only if visibleStart > 0
  • Otherwise, masks it along with digits.

Examples: "+2348123456789" (visibleStart=2, visibleEnd=3) → "+81****6789" "+2348123456789" (visibleStart=0) → "*******6789"

Parameters :
